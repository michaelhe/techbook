回顾2011，展望2012！

发表于 2017-03-25 | 分类于 life

起初，我把这个博客的定位在技术笔记上，后来发现这个后台实在是太难用了（相对与mediawiki来说），慢慢的，写的就少了。更多的时候，把笔记都写在私人的evernote上面了，时间管理的东西就用doit.im了。博客也慢慢的变成了一片荒草地，没办法，现在感觉时间太少了，以至于更新一篇都如此的难！下面就来记流水帐了…

2011

在这一年，对于我个人来说是不平凡的一年，几个关键字可以归纳一下。离职&入职、无方向=>互联网、RouterOS=>Linux、靠谱的公司&靠谱的老板&给力的导师。

离职&入职

5月份上半月，在拿到剩下的一半年终奖以后，我选择了离职。离开了我职业生涯的第一个公司xx”一家看似技术型的公司”。可能我在这里的职位“网络工程师”太尴尬了，所以在被闲置了大约半年的时间以后，我选择了离开，因为实在是找不到继续呆下去的理由了。从心底里讲，我还是挺感激它的，至少它给了我在深圳生活下去最基本的物质保障，给我上了职业生涯的第一课，在这里认识了一大群跟我一样青涩的大学毕业生们，虽然现在还呆在这里的已经不足一半了。直到现在，我们还会在群里侃侃自己的生活，有时间找几个朋友小聚，非常难忘的职业生涯的起点。

5月份下半月，上天让我非常有缘分的加入了xx，这个成立刚过半年的创业公司，从5月份到现在半年多的时间，我认为是我自己在毕业以后成长最快的半年，学的知识也是最多的半年，收获也是最多的半年，所以我非常感谢xx，这个温馨而美好的大家庭。

无方向=>互联网

毕业以后，才明白现实与理想的差距实在是太大了。有人说毕业后的第一份工作非常重要，而我不这样认为，大多数的同学都是在自己不知道自己喜欢什么的情况下选择了一份工作，或者说是被迫选择了一份工作（比如说我），当工作一段时间以后，也许有些同学还是不知道自己要什么，索性就把这个工作当作自己的职业一直继续下去了,当明白自己的想要的东西以后，又舍不得自己毕业后所经历的。于是我就武断的下了一个结论：毕业以后换的第一份工作是最重要的。有些同学换工作仅仅为了薪水 的增加，如果是这样子的话，离下一次离职也不远了。我觉得大多数同学换了一份工作以后都会把它当作毕生的事业来做，所以换工作一定要谨慎。

很小的时候，我比同龄的孩子更早的接触了互联网，选择互联网算是小时候好奇的一种回归吧。

RouterOS=>Linux

以前我的blog大部分是关于网络部分的，最典型的是routeros的东西写的比较多，而且大部分是通过讲原理的方式来让自己对网络的路由交换原理理解的更加透彻，而且我很不喜欢将一件很简单的事情重复的去做很多遍，比如在交换机上配置vlan这种低脑力劳动，高体力劳动，我更愿意让计算机去帮我完成，所以我喜欢上了脚本，大家可以看到后面routeros的很多东西我都是写的脚本在上面。慢慢的，我发现网络只能解决一些基础问题，按照TCP/IP的分层理论的话，它解决不了第7层的问题，给你网络设备，你只在那边ping来ping去是没有任何意义的，所以我发现了网络设备一定要跟网络服务器结合在一起用，才是现代互联网的归属。而我正在转型，从关注基础服务多一点，到关注应用服务多一点，老实地说，玩linux的成就感比玩网络的成就感要大很多。也许这也是我的导师所说的从偏硬在向偏软转换吧。

靠谱的公司&靠谱的老板&给力的导师

我很幸运在加入xx以后，逐渐感受到这是一个靠谱的公司，认认真真在做事情的一家公司，老板基本上是每天最晚回家的那一个人，你还能不说这是一个靠谱的老板么？在我入职还不到半年的时候拿到公司的期权，而且在今年的年会上，老板还说要给这些期权分红，你还能hold住吗？最重要的是，到了公司以后，有一个非常给力的导师，无论在技术上还是解决问题的思路、态度以及心态上，都给了我很多的帮助，这将是我在今后的人生道路上的宝贵财富。我只想轻轻的对公司，对老板，对导师说：“谢谢！我不会辜负你们对我的期望！”

2012

无论2012是否是世界末日，我是否能拿到船票，都希望自己能完成下面几件事情：

  1. 用足够多的时间去关心家人，关心朋友，关心亲爱的她；
  2. 努力的工作，尽职尽责的完成老板交给我的每一项任务；
  3. 去祖国的大好江山（非湖北、广东）的某一个地方溜一圈；
  4. 未完待续……

（ps:因为某种原因，我略去了公司名称，请见谅！）
