说说我之前的工作（一）

发表于 2017-03-25 | 分类于 work

很久没写东西了，以后的工作也会多多少少的跟文字沾上一点边。原因之一是锻炼自己的写作能力，第二也想重新反思下工作这几年的沉淀，所以有了这篇文章。能写多久，会写多少，看缘分了。

网络工程师@某网络设备厂商

1. 跟我一起入职的网络工程师现在都在干嘛？

毕业已经看起来是很久远的事情了。已经不太记得为什么工作的第一份工作选择网络，事实上在工作一年半左右的时间，大家就纷纷转岗了，大部分去做了网络设备的测试或是售前咨询的岗位。

说实话，这是一个门槛不是很高的行业，发展前景也不乐观的一个职业，如果你是毕业生看到这篇文章，请谨慎选择。

2. 如何做好这份工作？

想要做好这份工作，我是最没有发言权了，因为我本身就没有做的太好。一个是公司对你工作岗位定位，另外一个是个人兴趣导向。使得我自己在这个方向上没做出什么成绩，不过，我看到身边的一些做的好的同事，他们一些特有的素质。

  1. 专业性，理论知识需要打牢固，如7层网络模型各层的协议你了解了多少
  2. 实践，有了理论就需要实践，没有现成的机器需要装模拟器，装虚拟机
  3. 认证，各种思科认证CCNA/CCNP/CCIE，华为认证你到底能到哪一层
  4. 态度，态度决定一切，在任何时候都会适用

3. 这一年，我都做了啥？

公司不是学校，不会给你钱还让你学东西，作为职场的你，一定要清楚，你为公司创造了多少价值？

  1. Helpdesk，负责了公司电脑和网络的维护工作
  2. 新园区的建设，网络规划和实施，其中还有部分布线
  3. 学会了一个叫Routeros的技能，现在已经在两所学校实施了这一成果
  4. 新园区的自动化配置建设（现在看来就是企业内部的一个cmdb）

最后一个做的事情现在看来最有意义，让接下来的维护的一个同学转行做了php开发，也有幸让我进入了互联网行业。

4. 这份工作让我学到了什么？

一个懵懂的学生，初入职场，没有经历过什么尔虞我诈，没有勾心斗角，一心只为学技术。

  1. 收获了到现在还偶尔还会出来吃吃东西的好朋友
  2. 收获了到现在还会用到的网络技巧
  3. 收获了很好的自学的习惯，凡事都要自己去摸索
  4. 收获了工作的态度，凡事答应了就要做到，做到就要做好
  5. 还收获了在深圳活下去的第一笔收入
